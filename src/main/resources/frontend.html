Equation Solver

This is java implementatino of algorithm described in Similarity-Based Constraint Solving Modulo Commutativity: A Unified Framework


Result:

Input Syntax:

  • A character followed by "_o" is an ordered function symbol.
  • A character followed by "_u" is an unordered function symbol.
  • An alphabet character without "_o" or "_u" is a variable.
  • Equations must be in Disjunctive Normal Form (DNF).
